Please set the environment using the following command before compile java visualization program.

export CLASSPATH=$CLASSPATH:your_path/jfreechart-1.0.19.jar:your_path/jcommon-1.0.23.jar 

your_path is your path to the two jar libs and the two jar libs is in the lib directory.


The command to use the java visualization program.

java HzSerial -comm serial@/dev/ttyUSB0:telosb //run java
